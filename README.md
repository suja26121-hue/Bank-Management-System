# 🏦 Bank Management System (C++)

## 📌 Overview

This is a console-based Bank Management System developed using C++.
The application simulates basic banking operations and securely stores customer records using file handling.

It is designed with a clean menu-driven interface and follows object-oriented programming principles.

---

## 🚀 Features

* Create new bank accounts
* Deposit money into customer accounts
* Withdraw money securely
* Balance inquiry system
* Display all customer records
* Persistent data storage using binary files
* Clean banking-style console UI
* Input validation for safer transactions

---

## 🛠️ Technologies Used

* C++
* File Handling (Binary Files)
* Object-Oriented Programming (OOP)

---

## 💡 Key Concepts Used

* Classes and Objects
* Encapsulation
* File Streams (`ifstream`, `ofstream`, `fstream`)
* Menu-driven programming
* Data validation
* Record management

## ▶️ How to Run

### Using VS Code:

1. Install MinGW C++ Compiler
2. Open project folder
3. Compile:

```bash
g++ main.cpp -o bank
```

4. Run:

```bash
./bank
```

---

### Using Dev C++:

1. Open `main.cpp`
2. Click **Compile & Run**

```text

```

This allows records to remain saved even after closing the application.
